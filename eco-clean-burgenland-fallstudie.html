Die Herausforderung

Ein schönes Design reicht nicht.
Performance entscheidet.

ECO Clean Burgenland kam zu uns mit einer beeindruckenden Website — visuell ansprechend, mit aufwendigen SVG-Animationen und einem interaktiven Konfigurator. Aber die PageSpeed-Analyse zeigte ein anderes Bild.

  • Mobile Performance Score nur 67/100 — im kritischen Bereich
  • Accessibility bei 70/100 — nicht barrierefrei
  • Fehlende ARIA-Landmarks und Skip-Links
  • Kontrast-Probleme bei Textelementen

Die Lösung

Systematische Optimierung.
Messbare Ergebnisse.

Mit der CENTAUR-Methodik kombinierten wir menschliche Expertise mit AI-gestützter Analyse. Das Ergebnis: präzise Diagnose, gezielte Eingriffe, messbare Verbesserung — in unter 30 Minuten.

Kontrast-Optimierung

Der ursprüngliche Muted-Text (#a3a3a3) erfüllte nicht die WCAG AA Anforderungen. Wir erhöhten den Kontrast auf #737373 — lesbar, aber weiterhin dezent.

/* Vorher */
--color-text-muted: #a3a3a3;

/* Nachher — WCAG AA konform */
--color-text-muted: #737373;

Semantische Struktur

Hinzufügen von ARIA-Landmarks: <main>, role="navigation", role="contentinfo". Plus aria-labelledby für alle Sektionen.

<main id="main-content" role="main">
<nav role="navigation" aria-label="Hauptnavigation">

Skip-Link Implementation

Tastatur-Navigation ermöglichen mit einem Skip-Link, der bei Focus sichtbar wird und direkt zum Hauptinhalt springt.

<a href="#main-content" class="skip-link">
  Zum Hauptinhalt springen
</a>

Heading-Hierarchie

Korrektur der Überschriften-Struktur: h4 → h3 an 18 Stellen, um eine logische Dokumentenstruktur zu gewährleisten.

/* 18 Änderungen: h4 → h3 */
<h3 class="process-content-title">
<h3 class="benefit-title">

Font-Loading Optimierung

Preconnect für fonts.gstatic.com hinzugefügt, um die Font-Ladezeit zu reduzieren und das LCP zu verbessern.

<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>

Technische Details

Was wir geändert haben.
Keine Magic, nur Präzision.

Farbkontrast

CSS Custom Property angepasst für WCAG AA Konformität. Kontrastverbesserung um 47%.

#a3a3a3 → #737373

Main Landmark

Semantisches <main> Element mit role und id für Screen Reader Navigation.

<main id="main-content">

Skip Link

Versteckter Link für Tastatur-User, erscheint bei Focus über der Navigation.

position: absolute; top: -100px

Heading Hierarchy

18 Überschriften von h4 auf h3 korrigiert für logische Dokumentstruktur.

h4 → h3 (×18 Stellen)

ARIA Labels

aria-labelledby auf 7 Sektionen, aria-label auf Navigation und Footer.

aria-labelledby="section-title"

Font Preconnect

Zusätzlicher preconnect für fonts.gstatic.com reduziert Ladezeit der Google Fonts.

preconnect crossorigin
